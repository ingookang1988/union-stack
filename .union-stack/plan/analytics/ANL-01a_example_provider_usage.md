# [ANL-01a] (예시) provider 사용량·근거 분석

## 분석 (가공 더미)
- (예시) 유입 트래픽의 provider 분포: Google 68% / Apple 24% / 기타 8%.
- (예시) 기타 8%는 6개 provider로 파편화 → 유지비 대비 효익 낮음(근인: 운영 복잡도).

## 발견 (→ 결정 근거)
- (예시) 우선 2종(Google·Apple)만 지원해도 커버리지 92% → `[PLAN-01a]`의 범위 축소를 뒷받침.

## 산출 (→ 뒷받침)
- → `[PLAN-01a]` 의 결정을 뒷받침(이 분석이 근거로 들어간 plan). 같은 계보 `01a`.
