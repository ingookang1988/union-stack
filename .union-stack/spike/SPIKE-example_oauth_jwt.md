# (예시) SPIKE: JWT vs 서버 세션

## 가설 (가공 더미)
- (예시) JWT로 무상태 세션이 우리 규모에 충분한가?

## 시도 / 관찰
- (예시) 토큰 갱신 흐름 PoC. 회전 토큰 복잡도 확인.

## 출구 (셋 중 하나로 종료)
- [ ] 성공 → `PLAN-*`로 승격 (+ `MTG-*`에 근거)
- [ ] 실패 → `LSN-*`로 증류
