# Handoff → 다음 세션

## 1. 세션 요약 (1~3줄)
- (예시) 소셜 로그인 OAuth 셋업 착수. NextAuth 골격 배치까지 완료.

## 2. 변경 위치 (ID 목록 — Upward Fetching 진입점)
- `[WO-01a-1]` 설정 완료
- `[FLOW-01a]` 데이터 리니지 계약 갱신
- 코드: (예시) `/src/auth/*`

## 3. 다음 작업 (단일 진입점)
- → `[WO-01a-2]` user DTO 생성부터 시작.

## 4. 미해결 / 주의
- (예시) provider enum이 `[CON-01]`과 아직 정합되지 않음 — 먼저 확인할 것.
- (예시) `[LSN-01a]` 함정(provider 검증 누락) 재발 주의.

## 5. 검증 상태
- (예시) 단위 테스트 통과, E2E 미실행.
