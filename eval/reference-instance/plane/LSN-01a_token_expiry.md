# [LSN-01a] 토큰 사용 전 만료 검사 누락 (반복 3회)

**증상:** 토큰을 `getToken()`으로 꺼내 *그대로 사용*해, 만료된 토큰으로 호출 → 간헐적 401/auth 실패.
**계보:** `01a` (providerGateway 토큰 경로). 과거 3개 세션에서 반복 관찰됨.
**처방(한 줄):** 토큰을 쓰기 *전에* 반드시 `tokenStore.isExpired(key)`로 확인하고, 만료면 재발급/갱신 후 사용.
헬퍼 `isExpired()`는 이미 제공된다 — 새로 만들지 말 것. → 상세 `[EVD-01a-3]`.

> 이 경고는 코드 작성 *전에* 읽혀야 가치가 있다(시험 전 오답노트). Upward Fetching이 `01a` 계보 진입 시 선주입.
